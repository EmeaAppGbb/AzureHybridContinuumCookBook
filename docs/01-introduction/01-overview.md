# Overview

## Abstract

The Azure Hybrid Continuum CookBook is a comprehensive guide to designing, building, and operating hybrid and sovereign cloud architectures using Microsoft Azure technologies. It covers the full spectrum — from cloud-native workloads running in Azure public cloud, through hybrid deployments connected via Azure Arc and Azure Local, to fully disconnected and air-gapped environments for sovereign and regulated scenarios.

This guide provides architectural patterns, decision frameworks, and a concrete reference implementation showing how an enterprise application can move through the entire continuum.

## Why This Guide Exists

Organizations face increasing pressure to:

- Meet data residency and sovereignty requirements across jurisdictions
- Maintain operational resilience in connectivity-constrained environments
- Execute cloud exit strategies when regulatory or strategic needs demand it
- Run consistent workloads across diverse infrastructure — cloud, edge, and on-premises

Azure provides a rich set of technologies to address these challenges, but the architectural decisions required to navigate this landscape are complex. This guide distills best practices, patterns, and concrete examples grounded in official Microsoft documentation.
