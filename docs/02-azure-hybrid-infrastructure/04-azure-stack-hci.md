# Azure Stack HCI

## Introduction

Azure Stack HCI is the hyperconverged infrastructure (HCI) solution that runs on validated hardware and integrates deeply with Azure. Understanding its role is important for context, as Azure Local builds on and evolves from Azure Stack HCI.
