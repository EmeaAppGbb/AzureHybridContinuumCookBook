# Part 9: Appendix

Supplementary reference materials for the Azure Hybrid Continuum CookBook. This appendix includes a comprehensive glossary of terms, a mapping of Azure PaaS services to their on-premises alternatives, and curated additional resources for continued learning.

## Contents

| Document | Description |
|----------|-------------|

## How to Use This Appendix

- **Glossary**: Reference for understanding terminology related to hybrid cloud, sovereignty, Azure Arc, Azure Local, and compliance frameworks
- **Service Mapping**: Practical reference for architects planning cloud exit or hybrid deployments—shows what to use on-premises when a cloud PaaS service isn't available
- **Additional Resources**: Starting points for deeper exploration including Microsoft Learn paths, reference architectures, GitHub repositories, and community content
