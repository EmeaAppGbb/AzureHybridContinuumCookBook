# Glossary

## Introduction

Definitions of key terms and concepts used throughout the Azure Hybrid Continuum CookBook.

## Topics to Cover

- [ ] Comprehensive glossary covering:
  - **Air-Gapped**: An environment with no network connectivity to external systems
  - **Azure Arc**: Management plane that extends Azure to any infrastructure
  - **Azure Local**: Azure infrastructure platform deployed on-premises
  - **Azure Stack HCI**: Hyperconverged infrastructure solution that runs on validated hardware
  - **Azure Stack Hub**: Integrated system for disconnected Azure deployments (PaaS-capable)
  - **Cloud Exit / Cloud Repatriation**: Moving workloads from public cloud to on-premises
  - **Confidential Computing**: Technology protecting data while it is being processed
  - **Data Residency**: Physical location where data is stored
  - **Data Sovereignty**: Legal authority over data, including who can access it
  - **ExpressRoute**: Dedicated private connection from on-premises to Azure
  - **Hybrid Continuum**: Spectrum of deployment models from public cloud to disconnected
  - **Landing Zone**: Pre-configured Azure environment with governance, security, and networking
  - **Operational Sovereignty**: Control over operations and infrastructure
  - **Sovereign Landing Zone (SLZ)**: Landing zone variant with sovereign controls
  - **Software Sovereignty**: Independence from cloud provider software dependencies
  - **Trusted Execution Environment (TEE)**: Hardware-protected processing environment
  - **Zero Trust**: Security model that verifies every access request regardless of location
